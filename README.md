# Consulta de CEP

Projeto desenvolvido como desafio de estágio para praticar HTML, CSS,
JavaScript, consumo de API e Git/GitHub.

## O que o projeto faz

O usuário digita um CEP brasileiro e o sistema consulta a API ViaCEP
para exibir o endereço completo na tela, com validação de entrada e
tratamento de erros.

## Tecnologias usadas

- HTML
- CSS
- JavaScript puro
- API ViaCEP

## Como rodar

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Digite um CEP com 8 números e clique em Consultar.

## API usada

https://viacep.com.br/

## Aprendizados

- Estrutura semântica de HTML com formulários
- Estilização com CSS e responsividade com media queries
- Manipulação do DOM com querySelector e innerHTML
- Consumo de API com fetch e tratamento de JSON
- Controle de versão com Git e publicação no GitHub
